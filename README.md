# Midterm Project — Multi-Constraint Portfolio Allocation  

**Name:** Mateo Zeppieri  
**Field:** Professional Baseball Analytics  

## Project Overview  
This project analyzes how to allocate a fixed payroll budget across professional baseball player acquisition options using a multi-constraint optimization model. The objective is to maximize projected team performance (wins added) while satisfying budget, roster balance, and risk constraints.

## Allocation Problem  
I modeled payroll allocation across nine player options grouped into three categories: Position Players, Starting Pitchers, and Relief Pitchers. The model applies a budget cap, minimum allocation requirements by category, and a concentration cap to ensure balanced and realistic roster construction.

## Tools Used  
- Python (PuLP, pandas, NumPy, matplotlib)  
- Linear programming optimization  
- Sensitivity analysis and scenario testing  

## Summary  
The model recommends allocating payroll primarily toward the highest-performing player options while maintaining roster balance and controlling financial risk. Sensitivity analysis, near-optimal alternatives, and stakeholder scenarios demonstrate that multiple strong solutions exist and that results depend most heavily on performance projections.
